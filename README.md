# Vanilla Outsider: Bat Ecology

![Version](https://img.shields.io/badge/Minecraft-26.1-green) ![License](https://img.shields.io/badge/License-GPLv3-blue)

> *Some call them pests. We call them the night shift.*

**Bat Ecology** transforms the Minecraft bat from a useless ambient particle into a complex, integrated part of the ecosystem. It introduces day/night roosting cycles, swarm intelligence, and a purpose for their existence: farming utility via Guano and Pollination.
